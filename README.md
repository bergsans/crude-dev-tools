# CrudeDevTools

CrudeDevTools is a minimalistic interface library for console applications. It contains

- A figlet wrapper
- A boxifyer
- A text formatter (*allows colors, bold)
- A object pretty-printer

## Setup
```
$ npm install crude-dev-tools
```
